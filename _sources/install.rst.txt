
Installation
############

Requirements
============

- python 

- pymatgen

- ase

Installation
============

1.To use PyCSP, first install pymatgen and numpy using pip:

.. code:: console

   pip install numpy
   pip install pymatgen

2.Install PySCP

.. code:: pyth

   pip install pycsp

In addition, you can also visit `pypi`_ to download.

.. _pypi: https://pypi.org/project/pymatsci/
